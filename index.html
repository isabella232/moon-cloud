How it works

  • You get a completely dedicated cluster in one of supported cloud platforms: AWS, Google Cloud or Microsoft Azure.
  • We always run this cluster in the geographic region near you, so network connectivity is lightning fast.
  • Running in Kubernetes Moon Cloud has autoscaling feature, i.e. shrinks and grows depending on how many browsers you need. Because of that you only pay for computing resources you really consume.
  • You can use all popular browser automation frameworks: Selenium, Playwright, Puppeteer and Cypress simultaneously.
  • Moon Cloud relies on cloud platform networking features and can access your workstation, corporate network server or another cloud project.

Pricing

Because of pay as you go model Moon Cloud pricing model differs from traditional cloud platforms. At the end of each month you pay regular cloud platform bill plus a fixed commission of 25% for cluster maintenance.
